🌐 News Headlines Web Scraping & Visualization

📌 Overview

This project demonstrates web scraping and data visualization using Python.
It extracts real-time news headlines from a website and analyzes trending topics using word frequency.

---

🎯 Objectives

- Extract data from a live website
- Parse HTML content using BeautifulSoup
- Store structured data in a DataFrame
- Perform basic text analysis
- Visualize trends using a bar chart

---

🗂️ Dataset

- Source: https://news.ycombinator.com/
- Data Collected:
  - News Headlines
  - Article Links

---

🛠️ Tools & Technologies

- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

📊 Process

1. Sent HTTP request to fetch webpage
2. Parsed HTML using BeautifulSoup
3. Extracted headlines and links
4. Stored data in a DataFrame
5. Performed text cleaning and preprocessing
6. Counted word frequency
7. Visualized top words using bar graph

---

📈 Key Insights

- Frequently occurring words indicate trending topics
- Headlines reveal focus areas like technology and startups
- Visualization helps quickly identify popular themes

---

📷 Output

- CSV file containing scraped data
- Bar graph showing most common words in headlines

---

📌 Conclusion

This project shows how web scraping combined with visualization can transform raw web data into meaningful insights.
